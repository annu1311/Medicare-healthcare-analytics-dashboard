# Medicare & Medicaid Healthcare Analytics Dashboard (Power BI)

An interactive Power BI dashboard analyzing healthcare accessibility, service utilization, and provider availability across U.S. states from 2020 to 2024.  
Designed to support data-driven decisions for healthcare performance evaluation.
